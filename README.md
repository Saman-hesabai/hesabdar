# Hesabdar v0.5 AI Glass

نسخه 0.5 حسابدار:

- ظاهر شیشه‌ای‌تر و حرفه‌ای‌تر
- کارت‌های آماری فشرده‌تر
- منوی پایین برای دسترسی سریع
- دکمه شناور دستیار حسابدار
- پنجره دستیار متنی
- فرمان صوتی آزمایشی با Web Speech API در مرورگرهای سازگار
- پاسخ صوتی دستیار با Speech Synthesis
- گزارش امروز، بدهکارترین مشتری، مانده مشتری
- ثبت مشتری، ثبت نسیه، ثبت پرداخت، سطل زباله

نکته: فرمان صوتی کامل به پشتیبانی مرورگر بستگی دارد. برای هوش مصنوعی واقعی و مکالمه‌ای، در نسخه بعدی باید Supabase Edge Function + OpenAI API اضافه شود.
