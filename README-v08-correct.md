# حسابدار v0.9 درست

مبنای این نسخه فایل `0.7 dorost.zip` است و اتصال فروشگاه و دیتابیس نسخه سالم 0.7 بدون تغییر حفظ شده است.

اصلاحات:
- ویرایش تراکنش با فرم واضح در بالای تاریخچه، پیام موفقیت و خطای واقعی
- پنل کامل مدیریت کارکنان برای مدیر اصلی
- افزودن، فعال/غیرفعال‌کردن، حذف و تغییر دسترسی کارمند
- دستیار فارسی با پشتیبانی از اعداد رقمی و واژه‌ای مانند «ده هزار»
- ثبت توضیحات اقلام همراه تاریخ و ساعت

در صورت نبود ستون‌های مدیریت کارکنان، فایل `supabase-v08-correct.sql` را در پروژه صحیح Supabase اجرا کنید.
